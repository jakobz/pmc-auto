Bookmarklet to copy Time Journal report from previous week

  1. Drag-n-drop this link to your bookmarks panel (or bookmark it another way):
    Fill PMC
  2. Open the Time app
  3. Click the bookmarked link
  4. Click Save
Works only in 2-weeks view. Copies left week values to the right week. Skips holidays/vacs on both left and right week while copying. If left week has more work days than right, the last day of the left week will be duplicated Source: 
document.querySelectorAll('.nested-data.activities-container .table-right-part').forEach(part => {
   let leftDays = part.querySelectorAll('.table-week.first-week .table-activity-cell.cell input:not(.not-workday)');
   let rightDays = part.querySelectorAll('.table-week.last-week .table-activity-cell.cell input:not(.not-workday)');
   rightDays.forEach((rightDay, idx) => {
     rightDay.value = leftDays[Math.min(idx, leftDays.length - 1)].value;
     var event = new Event('blur');
     rightDay.dispatchEvent(event);
   })
})
